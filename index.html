软件开发学习计划

通过系统自学，掌握软件开发全流程，成为具备项目能力的开发者

一、学习总目标

通过系统自学，你最终应具备以下能力：

  • 理解软件开发全过程（需求、设计、编码、测试、上线、维护）
  • 能独立做出一个中小型完整项目（例如网站、App 或桌面软件）
  • 掌握主流工具与工程思维（Git、项目管理、版本控制、协作流程）
  • 能看懂专业文档、参与团队协作，或进入自由职业/自媒体开发路线

二、学习前的准备

硬技能准备

类别 说明 建议工具
电脑环境 一台能流畅运行 VS Code、浏览器、Git 的电脑 推荐 Win/Mac/Linux 都可
编辑器 代码编辑器 Visual Studio Code
版本控制 学会保存和回退代码 Git + GitHub（或 Gitee）
运行环境 用于跑网页或脚本 Node.js（前端方向）/ Python 环境（后端方向）

三、学习路线（阶段划分）

阶段 1：计算机与编程入门（1~2个月）

目标：理解计算机和代码的基本概念，不死磕数学。

学什么：

  • 计算机原理基础（了解 CPU、内存、文件系统）
  • 编程思维（条件、循环、函数、模块）
  • 入门语言：推荐 Python（语法简单，适合工程思维）

推荐资源：

📺 视频：B 站「程序员鱼皮」：《Python 入门教程（零基础）》、YouTube：「freeCodeCamp Python for Beginners」

📚 书籍：《Python编程：从入门到实践》、《Head First Programming》

阶段 2：前端开发基础（2~3个月）

目标：能做出网页并理解"用户界面 + 逻辑 + 交互"。

学什么：

  • HTML + CSS（网页结构和样式）
  • JavaScript（交互逻辑）
  • 浏览器与调试工具
  • 学会用 VS Code 实时预览网页

推荐资源：

📺 视频：B 站「黑马程序员」：《前端入门到精通》、YouTube：「Web Dev Simplified」HTML/CSS/JS 系列

📚 书籍：《Head First HTML & CSS》、《JavaScript DOM 编程艺术》

阶段 3：前端工程化与项目实践（2~3个月）

目标：从网页变成真正的"软件项目"。

学什么：

  • Git + GitHub 项目管理
  • Node.js & npm（包管理工具）
  • 前端框架：React（现代软件工程标配）
  • API 调用与数据交互

推荐资源：

📺 视频：B 站「Coderwhy」React 全家桶教程、YouTube：「Traversy Media」React Crash Course

📚 书籍：《React 进阶之路》、《深入浅出 Node.js》（部分章节）

阶段 4：软件工程核心思想（2个月）

目标：理解"做项目"的完整流程。

学什么：

  • 软件生命周期：需求分析 → 设计 → 编码 → 测试 → 部署 → 维护
  • 版本控制（Git Flow）
  • 测试与质量保障（单元测试）
  • 项目管理（Scrum、敏捷开发）

推荐资源：

📺 视频：B 站「极客时间」或「阿里云开发者」的软件工程系列讲解

📚 书籍：《软件工程：实践者的研究方法》、《人月神话》、《敏捷软件开发：原则、模式与实践》

阶段 5：综合实战项目（持续进阶）

目标：从"学生"变"开发者"。

可选方向：

  • ✅ 个人网站 / 博客系统（前端+后端）
  • ✅ 小岛周末导航页升级版（加用户系统、数据存储）
  • ✅ 旅行相册App / 备忘录App
  • ✅ 公众号 + 静态网页整合项目

实践建议：

  • 用 GitHub 建立公开项目仓库
  • 每周写一份开发日志
  • 尝试把作品部署上线（Netlify、Vercel、Coding Pages）
